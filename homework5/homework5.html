Steady State Diffusion

This can be solved using Fick’s First law of diffusion: J = -D \frac{dC}{dx}

This can be rearranged for the concentration gradient and solved for the high-pressure concentration \frac{dC}{dx} = -\frac{J}{D}

\Delta C = C_{\text{low-pressure}} - C_{\text{high-pressure}} = - \frac{J}{D}\Delta x

C_{\text{high-pressure}} = C_{\text{low-pressure}} + \frac{J}{D}\Delta x

Now plugging in the given values: C_{\text{high-pressure}} = 0.5\,\text{kg/m}^3 + \left(0.002\,\text{m}\right) \frac{2.48\times10^{-8}\ \text{kg/m}^2\text{s}}{4.50\times10^{-11}\ \text{m}^2\text{/s}} = 1.60\,\text{kg/m}^3
